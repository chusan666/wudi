## 🎯 核心功能演示

### 视频解析流程
1. 用户输入视频链接
2. 系统自动识别平台（小红书/抖音/B站/快手）
3. 调用对应解析器提取视频信息
4. 展示视频标题、作者、封面、互动数据等

### 视频分析流程
1. 点击"拆解分析"按钮
2. 系统进行多维度分析
3. 通过4个标签页展示：
   - 📊 综合概览
   - 📈 互动数据
   - 📝 内容分析
   - 💡 优化建议

## 🚀 快速启动

### 方式1: 一键启动（推荐）
```bash
./start.sh
```

### 方式2: 手动启动
```bash
# 后端
cd backend
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python app.py

# 前端（新终端）
cd frontend
npm install
npm run dev
```

### 方式3: Docker部署
```bash
docker-compose up -d
```

## 📝 使用说明

1. **启动服务**
   ```bash
   ./start.sh
   ```

2. **访问应用**
   - 打开浏览器访问: http://localhost:5173

3. **使用流程**
   - 粘贴视频链接（支持小红书/抖音/B站/快手）
   - 点击"解析"按钮
   - 查看视频信息
   - 点击"拆解分析"按钮
   - 查看多维度分析结果
